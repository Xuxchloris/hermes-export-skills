# Compliance Notes

This repository is a workflow and documentation kit. It does not provide customer databases or send outreach automatically.

Recommended boundaries:

- Keep all outbound email as human-reviewed drafts.
- Keep all quotations as human-reviewed drafts.
- Do not invent contact names, buyer intent, certifications, prices, freight, or delivery dates.
- Record the source of each prospect and personalization claim.
- For GDPR and similar regimes, include a lawful basis assessment and opt-out language.
- Use approved business sources and local anti-spam rules.
- Never commit real `.env` files, API keys, customer price lists, or private lead lists.

Operational rule:

```text
Research and draft with Agent. Confirm and send by human.
```
