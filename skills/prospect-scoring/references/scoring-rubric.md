# Scoring Rubric

| Dimension | Points | Guide |
| --- | ---: | --- |
| Product fit | 40 | Same category, adjacent category, or clear application overlap |
| Channel value | 20 | Importer, distributor, wholesaler, brand, retail chain |
| Decision-maker access | 15 | Named role, department email, generic contact only |
| Target-market value | 10 | Matches configured country, language, and market segment |
| Website quality | 10 | Active website, catalog, contact page, recent updates |
| Risk adjustment | -10 | Weak evidence, unrelated business, compliance issue |
